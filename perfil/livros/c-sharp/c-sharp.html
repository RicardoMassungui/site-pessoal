Capítulo 1 A história do C #
C # continua a evoluir e melhorar a cada versão principal da linguagem. Como o C #
equipe está inovando e adicionando recursos ao C #, seus membros compartilham seu processo de pensamento em torno
o design do C # com a comunidade.
Se você for ao repositório dotnet / Roslyn no GitHub , você verá a linguagem detalhada
status de recursos, bem como recursos que a equipe C # está considerando para os próximos lançamentos. Vocês
também pode ver um histórico C # no Microsoft Docs . No momento em que este livro foi escrito, de particular interesse são os
recursos planejados em torno do C # 9. Posteriormente neste livro, daremos uma olhada no que está planejado para
C # 9.
As ferramentas de construção C # padronizarão a versão do idioma para a versão principal mais recente. A partir do C # 7.0,
no entanto, os desenvolvedores começaram a ver mais lançamentos pontuais na forma de versões C # 7.1, C # 7.2,
e C # 7.3.

projeto.
